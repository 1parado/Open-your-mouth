# Web Design Notes

## Direction

- 保持原来的简洁、轻量、直白风格
- 页面以白底、浅灰边框和简单卡片为主
- 不使用强装饰背景、复杂动效或明显的品牌化视觉实验

## Typography

- 使用 `Inter`
- 标题直接、清楚，避免过多装饰性字体
- 正文强调可读性和轻量感

## Color

- 页面背景：接近白色的浅灰
- 文本主色：深灰
- 辅助文本：中灰
- 卡片和容器：白色背景 + 浅灰边框

## Layout

- 首页保持居中布局
- 功能入口继续用三张卡片承载
- 子页面使用统一的返回入口和简单内容区块

## Component Rules

- 卡片圆角适中，不要过度设计
- Hover 只保留轻微背景和边框变化
- 页面结构应优先清楚，而不是追求视觉记忆点

## Current Scope

- `/` 首页作为导航入口
- `/practice`、`/courses`、`/history` 保持同一套轻量页面结构
- 后续如果接真实业务页面，也继续沿用这套中性简洁风格
